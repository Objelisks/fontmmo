# fontmmo

mmo

## Running server
`npm run watch` (optional, needed if edits to clientside files are made)

`npm run server`

open `http://localhost:8080`

## Controls

keyboard: 

* wasd or arrow keys: movement
* h,j,k or z,x,c: abilities (1,2,3)

gamepad:

* left stick: movement
* X, A, B: abilities (1,2,3)
